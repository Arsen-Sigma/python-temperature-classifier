# 🌡️ Python Temperature Classifier

Мой второй проект на Python — консольная программа для определения погоды по температуре.

## 🚀 Возможности

- ❄️ Определение мороза
- 🧥 Определение холодной погоды
- 🌤️ Определение тёплой погоды
- ☀️ Определение жаркой погоды
- 🔥 Определение очень жаркой погоды
- 💡 Вывод полезных советов

## ▶️ Запуск

1. Убедитесь, что установлен Python 3.
2. Склонируйте репозиторий или скачайте его архивом.
3. Выполните команду:

```bash
python temperature_classifier.py
```

## 📋 Пример работы

```text
Введите температуру: 28

🌡 Температура: 28°C
☀️ На улице жарко.
💧 Совет: возьмите воду.
```

## 🛠️ Технологии

- Python 3
